# BoneForge Alternative CATS for Blender 5.0

Open-Blender BoneForge package for standard Blender users.

This repository is the non-exclusive Blender build. It should contain the CATS-compatible BoneForge 8.4.6 package, the complete CATS Material Combiner / UVToolkit toolset, and basic BoneForge/Mixamo rigging tools only. Matching the B4Artists version number does not make this the B4Artists-exclusive payload.

B4Artists-exclusive files must not be committed here.
